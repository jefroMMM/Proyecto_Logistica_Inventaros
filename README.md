# Proyecto_Logistica_Inventaros
Sistema enfocado en la logística y trazabilidad de inventarios, diseñado para ser intuitivo, seguro y eficiente.
Permite el control en tiempo real desde el ingreso de productos hasta su despacho.


# Problema
1)Actualmente existen dificultades en la gestión de inventarios, tales como:
2)Falta de certeza sobre existencias reales
3)Pérdida de productos
4)Problemas de ubicación interna en bodegas
5)Desbalances de stock
6)Retrasos en entregas
7)Mala toma de decisiones en compras
8)Pérdidas económicas

# Alcance
El sistema tiene en cuenta lo siguiente
1)Registro de bodegas y productos
2)Control de entradas y salidas
3)Comunicación entre bodegas
4)Historial de movimientos

# Stakeholders
1)Gerentes de comercios pequeños
2)Emprendedores
3)Encargados de bodega
4)Encargados de compras
5)Repartidores

# Definition of Success

1)El sistema será considerado exitoso si:
2)Permite visualizar inventario en tiempo real por bodega
3)Gestiona correctamente el abastecimiento
4)Genera alertas de stock mínimo
5)Reduce inconsistencias en inventarios
6)Genera reportes y posibles predicciones
